Hangman:
	Run game in terminal,
	Write the secret word.
	Guess letters contained in the word.
	
Cipher:
	Run in terminal with -d or -e. Terminal will display the encrypted message (Vignere cipher).
	
	'-d','--decrypt': Decrypt A Message(Spaces will not be included)
						
	'-e','--encrypt': Encrypt A Message(Spaces will not be included)
						
	'-k','--key': key to use for encryption/decryption of message (if not specified, default will be used)
