Detailed Description

Engine class, Used for loading resources such as textures and sounds.

Load texture files to sf::sprite pointers or sounds to sf::SoundBuffers.

Warning
MUST USE PROJECT IN RELEASE MODE FOR TEXTURES TO LOAD.
