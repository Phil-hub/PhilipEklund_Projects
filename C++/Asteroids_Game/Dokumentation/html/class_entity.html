Detailed Description

Entity class, This is the basic class for all objects on the screen.

This class is used for all objects displayed on the screen that use some kind of movement or collision detection.

Author
Philip EKlund
Version
1.0
Date
2021
Precondition
If you want a constant speed/rotation on the object (like an asteroid) set velocity and rotation at construction of object.
Warning
Improper use can crash your application!
